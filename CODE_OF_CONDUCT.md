# Code of Conduct

This project follows the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
(version 2.1).

By participating in this project — opening issues, submitting pull requests,
contributing to discussions, or interacting with maintainers and other
contributors — you agree to abide by its terms.

For the full text, see the canonical document at
[contributor-covenant.org](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting a violation

To report a violation, please
[open a private security advisory](https://github.com/PsycherosAI/Psycheros/security/advisories/new)
on the public repository. Advisories are confidential to the reporter and the
maintainers; use this channel for code-of-conduct reports as well as security
vulnerabilities. Maintainers will review the report and respond privately.
