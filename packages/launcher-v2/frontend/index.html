Psycheros · Data

      


      
Manage my data.

      

        Back ups go to your Downloads folder. Restore reads a zip
        you previously exported and overwrites my current state with it. Wipe
        and Re-init are destructive — wipe clears my memories/identity/vault;
        re-init reclones source and templates fresh identity.
      


      


      

        

          

            Back up
            
              Exports a snapshot of my entity data (memories, vault, identity,
              settings, generated images, and trusted local plugins) as a zip in
              your Downloads folder. Daemon must be running.
            
          

          
        


        

          

            Restore
            
              Reads a previously-exported zip and replaces my current data with
              what's inside it. Daemon must be running; I'll restart
              automatically when the import finishes. Restored plugins are
              executable local code and may contain secrets.
            
          

          
        


        

          

            Wipe entity data
            
              Clears my memories, identity files, vault docs, and database. The
              OS service registration stays — first-run will template fresh
              identity files on next launch. Cannot be undone unless you've
              backed up first.
            
          

          
        


        

          

            Re-init
            
              Stops the daemon, deletes my source clone + identity dir,
              uninstalls the OS service. You'll go through the first-run wizard
              again to re-bootstrap. Memories and vault content survive this —
              only identity is reset.
